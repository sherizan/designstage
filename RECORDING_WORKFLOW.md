## Key Features Implementation

### 1. Draggable Edges
- **Edge Detection**: 10px margin around selection border
- **Handle Types**: Corners (â—¯), Edges (â–¡)
- **Drag Types**: 
  - Corner: Resize both dimensions
  - Edge: Resize one dimension
  - Inside: Move entire region
- **Constraints**: Minimum 100Ã—100px, stay within screen bounds

### 2. Preset System
- **Three presets**: Square, Landscape, Portrait
- **Apply preset**: Maintains center position, changes dimensions
- **Custom mode**: Automatically switches when manually resized
- **Dropdown selector**: Live preview of dimensions

### 3. Video Gallery
- **Auto-thumbnail**: Generated from first frame using AVFoundation
- **Horizontal scroll**: NSScrollView with NSStackView
- **Hover effects**: Show delete button on hover
- **Click actions**: Play video or delete
- **Persistence**: Metadata in UserDefaults, files in ~/Movies/

### 4. Recording Quality
- **Resolution**: 2Ã— selected dimensions (Retina)
- **Codec**: H.264
- **Bitrate**: 6 Mbps (high quality)
- **Format**: .mov (QuickTime)
- **Cursor**: Included
- **Audio**: Not captured
